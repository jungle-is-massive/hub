01
Data sources

The dashboard pulls from two systems. HubSpot is the authoritative source for all pipeline and deal data. Synergist was previously used for the Now lane and deal counts but has been removed — it was causing double-counting. Any opportunity worth tracking must now be logged in HubSpot first.

HubSpot
CRM · Primary source
The sole source of truth for contacts, lifecycle stages, and deals. The dashboard queries HubSpot via a Supabase edge function proxy (hs-proxy) to avoid CORS issues. Two object types are used: Contacts (for Prospect Funnel Stages) and Deals (for all pipeline data). All deal queries — including the Now/Next/Future lanes, the pipeline value stat, and the deal source breakdown — are scoped to the Agency Pipeline only (2808364263).
Supabase
Database
Jungle's own database. Stores the funnel stage definitions — the list of boxes, their labels, order, and metadata (e.g. which maps to MQL, which to SQL). This means the funnel structure can be updated without touching code. Supabase also stores contacts imported from Dripify, and serves as the proxy layer that the dashboard's edge function routes through.
📌
The discipline: if an opportunity isn't logged in HubSpot, it won't appear on the dashboard. That's intentional. HubSpot is the forcing function — every deal needs to move through Opportunity → Pitch → Won or Declined within the Agency Pipeline for the numbers to be accurate.

The four stat cards

The large black cards at the top of the dashboard. Each one is a headline number with a different source.

Card Source What it counts Filter
Open Dealsstat-deals HubSpot Total open deals in the Agency Pipeline — excludes Closed Lost, Closed Won, and Declined Agency Pipeline only. All three closed stages excluded via separate NEQ filters.
Active Pipelinestat-pipeline HubSpot Total £ value of open Agency Pipeline deals, measured against a £6m target (3:1 on £2m revenue target) Same deal set as Open Deals. Progress bar shows % of £6m target covered. Cover ratio shown as X:1 against £2m.
Fresh Inquiriesstat-inquiries HubSpot Contacts recently created in HubSpot that have been BD-flagged Flagged via the BD star in the Fresh Enquiries tab of the opportunities dashboard.
Meetingsstat-meetings Google Calendar Upcoming meetings (next 30 days) and attended meetings (last 30 days) tagged #nbm or #netm in calendar Requires Google Calendar to be connected. Tags must appear in the event title.
03
Deal Opportunity Tracking

The five boxes immediately below the stat cards. These count deals in HubSpot — specifically in the Agency Pipeline. Each box maps to one or more deal stages in that pipeline.

Opportunities and Pitches pull from the active deals already loaded for the stat cards. Won, Lost, and Declined run separate parallel queries because all three closed stages are excluded from the main active deals fetch to keep pipeline numbers clean.

Below the boxes, the Now / Next / Future lanes break open deals into three horizons. All three pull from the same Agency Pipeline dataset — no other source is used.

Agency Pipeline only. There are multiple pipelines in HubSpot (Jungle Sales Pipeline, Agency Pipeline, Partnerships, IP Sponsorship). This section deliberately shows Agency Pipeline only (2808364263) — the one Coris is actively managing BD in. The others are not counted here.

04
Deal Source & Intermediary breakdown

Two HubSpot deal properties added on 1 May 2026 to enable channel-level attribution across the pipeline. Both appear on deal cards in the Now/Next/Future lanes and in the deal drawer.

Property Internal name Type Values
Deal Sourcedeal_source deal_source Dropdown (single) Cold Inbound, Client Referral, Staff Referral, Intermediary, Owned event, Speaking event, Paid/SEO, Organic social, PR
Intermediariesintermediaries intermediaries Dropdown (single) — conditional AAR, Ingenuity+, Creativebrief, Oystercatchers, Observatory International, Tuffon Hall, Auditstar, Individual Consultants, Masterclassing, The GO Network, Advertising Association
ℹ️
Conditional logic: the Intermediaries field is hidden in HubSpot unless Deal Source = Intermediary. This is enforced via HubSpot's conditional logic (Properties → Conditional Logic), created 1 May 2026. It prevents the field being filled in for non-intermediary deals, keeping attribution clean.

Deal Source breakdown bar — sits between the Deal Opportunity Tracking boxes and the Prospect Funnel Stages. Shows colour-coded pills, one per source, with a live deal count. Deals with no source tagged show as "Untagged" at reduced opacity. The Intermediary pill is clickable — tapping it slides open the Intermediary Pipeline comparison table below it.

Intermediary Pipeline — Actual vs Estimated — a table comparing actual intermediary-sourced deal value (from HubSpot) against the projected pipeline figures set in the intermediary plan. Estimated figures use the same 30% conversion rate as the intermediary plan dashboard. Each row shows a dual-layer progress bar: the grey layer is the estimate, the coloured layer is actual (green ≥100%, amber ≥50%, coral below). This section is collapsed by default and expands when the Intermediary source pill is clicked.

05
Prospect Funnel Stages

The six boxes in the bottom row. These count contacts in HubSpot by their lifecycle stage — not deals. A contact's lifecycle stage is a separate property to what pipeline their deal sits in.

The stage structure is defined in Supabase (the funnel_stages table) and the counts are fetched live from HubSpot on each page load, one request per stage in parallel.

ℹ️
Contacts, not deals. A contact tagged as Opportunity lifecycle stage is not the same as a deal in the Opportunity deal stage. Lifecycle stage tracks where a person is in the relationship — deals track active commercial conversations. Both matter, but they answer different questions.
Box HubSpot lifecycle stage Internal name Notes
01 · Lead Lead lead All contacts at the earliest lifecycle stage.
02 · MQL Marketing Qualified Lead marketingqualifiedlead Contacts who have met the MQL criteria (3+ content touches, 2+ email opens, fits ICP).
03 · SQL Sales Qualified Lead salesqualifiedlead Currently shows 0 — SQL criteria and the MQL→SQL process have not been defined yet. Accurate data, not a bug.
04 · Opportunity Opportunity opportunity Contacts with an active commercial conversation.
05 · Customer Customer customer Contacts whose deal has been won.
06 · Nurturing — — No HubSpot lifecycle stage mapped yet. HubSpot does support custom lifecycle stages — can be added in Settings → Objects → Contacts → Lifecycle Stage when ready.

07
Known gaps & next steps

Things that are intentionally deferred, pending a decision, or blocked on configuration.

  • SQL (box 03) shows 0 — There are currently no contacts tagged as Sales Qualified Lead in HubSpot. This is accurate — the criteria and process for moving contacts from MQL to SQL hasn't been defined yet. Needs a working definition before the number will move.
  • Nurturing (box 06) always shows — — No HubSpot lifecycle stage mapped yet. Can be added as a custom lifecycle stage in HubSpot (Settings → Objects → Contacts → Lifecycle Stage) when the team is ready to start tracking this cohort separately.
  • Deal Source not yet tagged on all open deals — As of 1 May 2026, 1 of 4 open deals has no Deal Source set (shows as "Untagged" on the dashboard). Needs to be completed in HubSpot for the source breakdown to be accurate.
  • Jungle Sales Pipeline not tracked — There are deals in the default Jungle Sales Pipeline which are not shown anywhere on this dashboard. Deliberate decision as of May 2026 — Coris is focused on Agency Pipeline. To be revisited once there is clarity on which pipeline is primary.
  • Won (box 09) — resolved — Closed Won stage added to Agency Pipeline on 1 May 2026 (ID: 5277858030). Now shows a live count from HubSpot.
  • Declined (box 11) — resolved — Declined stage added to Agency Pipeline on 1 May 2026 (ID: 5277854954). Runs as a separate query alongside Lost.
  • Double-counting in pipeline value — resolved — Dashboard previously pulled open deals from both Synergist (Now lane) and HubSpot (Next/Future lanes), causing inflated pipeline value. All deal data now comes exclusively from HubSpot Agency Pipeline.
